    1. `pub fn rgba_pixel_to_hsv(r: u8, g: u8, b: u8) -> (u8, u8, u8)` with `#[inline]` attribute.
       - Convert r,g,b to f32 in [0,1]: `rf = r as f32 / 255.0`, etc.
       - Compute `max = rf.max(gf).max(bf)`, `min = rf.min(gf).min(bf)`, `delta = max - min`.
       - Value `v = max`.
       - Saturation: if max == 0.0 then 0.0 else delta / max.
       - Hue: standard formula:
         - delta == 0.0 -> h_deg = 0.0
         - max == rf -> h_deg = 60.0 * (((gf - bf) / delta) % 6.0)
         - max == gf -> h_deg = 60.0 * (((bf - rf) / delta) + 4.0)
         - else -> h_deg = 60.0 * (((rf - gf) / delta) + 2.0)
       - Normalize: if h_deg < 0.0 then h_deg + 360.0.
       - OpenCV half-range: h = (h_deg / 2.0) as u8 (range 0-179), s = (s * 255.0) as u8, v = (v * 255.0) as u8.
       - Return (h, s, v).
       - Add Chinese doc comment: `/// 将 RGBA 像素转换为 HSV 颜色空间（OpenCV 半范围约定：H:0-179, S:0-255, V:0-255）`

    2. `pub fn count_matching_pixels(rgba: &[u8], rule: &ColorMatchConfig) -> usize`

       - Extract bounds: `let [h_lo, s_lo, v_lo] = rule.hsv_lower;` and `let [h_hi, s_hi, v_hi] = rule.hsv_upper;`
       - Use `rgba.chunks_exact(4)` to iterate pixels (4 bytes per pixel: R,G,B,A).
       - For each pixel chunk, call `rgba_pixel_to_hsv(pixel[0], pixel[1], pixel[2])` (ignore alpha per D-01).
       - Check if h >= h_lo as u8 && h <= h_hi as u8 && s >= s_lo as u8 && s <= s_hi as u8 && v >= v_lo as u8 && v <= v_hi as u8.

    **src-tauri/src/detection/validation.rs:**
    Implement one public function per D-13:

    ```rust
    use crate::models::ColorMatchConfig;

    /// 验证颜色匹配配置的阈值参数
    ///
    /// 规则（per D-13）:
    /// - min_pixels > 0
    /// - min_ratio in (0.0, 1.0]
    /// - hsv_lower <= hsv_upper per channel
    pub fn validate_color_match_config(config: &ColorMatchConfig) -> Result<(), String> {
        if config.min_pixels == 0 {
            return Err("最小像素数必须大于 0".to_string());
        }
        if config.min_ratio <= 0.0 || config.min_ratio > 1.0 {
            return Err("最小像素比例必须在 (0.0, 1.0] 范围内".to_string());
        }
        for ch in 0..3 {
            if config.hsv_lower[ch] > config.hsv_upper[ch] {
                return Err(format!(
                    "HSV 下界不能大于上界 (通道 {}): {} > {}",
                    ch, config.hsv_lower[ch], config.hsv_upper[ch]
                ));
            }
        }
        Ok(())
    }
    ```
